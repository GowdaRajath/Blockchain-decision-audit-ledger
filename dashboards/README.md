This dashboard combines multiple analytical views, including:
- Current decisions by lifecycle stage
- Average time between key lifecycle transitions
- Event chain length distribution
- Hash verification outcomes

Together, these views provide a consolidated snapshot of decision flow, audit completeness, and data integrity across the system.
